Repository

The sources are publicly available. For this purpose there is a CVS repository hosted by BerliOS:

User:anonymous
Host:cvs.extex.berlios.de
Location:/cvsroot/extex
Connection type:pserver
Module:ExTeX

If a password is requested send the empty string.

The following sample shows how to check-out the complete sources of &ExTeX; on a Unix box: 

   % cvs -d:pserver:anonymous@cvs.extex.berlios.de:/cvsroot/extex login
   % cvs -z3 -d:pserver:anonymous@cvs.extex.berlios.de:/cvsroot/extex co ExTeX

The commiting is restricted to members of the &ExTeX; team!

If you are interested to become a member of the ExTeX team write to extex@dante.de.
