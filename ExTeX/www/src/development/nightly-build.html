Nightly Build (Snapshot)

&ExTeX; is rebuild automatically once a day. The results are published on the &ExTeX; site. The following results can be found:

  • Installer
  • Javadoc (HTML)
  • Primitive and Syntax Reference (HTML)
  • User's Guide
  • Developer's Guide
  • Test Reports (HTML)

Snapshot Installer

The snapshot installer represents the last version from the repository. The least requirement for the snapshot is that the sources are compilable without error. If the sources do not compile then no new snapshot installer is published but the old one remains on the web site. Thus always a correctly compiled version can be accessed.

Note that the tests do not influence the publishing of a new snapshot installer.

Test Suite

&ExTeX; contains a test suite which is processed during the nightly build. The tests should be green for stable releases. For snapshots some tests might not be green.

The Test Reports provide some detailed insight into the results.
