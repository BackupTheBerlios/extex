Tasks

&ExTeX; relies on the help of many hands. To illustrate that differnt skills can aid in the development of &ExTeX; some tasks are described here. Those tasks are partially self-contained and can be performed independently in parallel to the main stream of development.

  • Writing an &ExTeX; debugger
  • Writing documentation
  • Testing primitives
  • Implement patgen
  • Maintain the Web pages

If you are interested in one of the tasks contact ExTeX Developers for details and coordination.
