This package contains primitives for ExTeX's TeX mode.

The Count Parameter \tracingrestores

The count register \tracingrestores contains controlls the logging of restore operations when the blocks are closed. The default implementation of ExTeX uses another mechanism. Thus this register has no effect at all.

This register is a preassigned count register and can be used as such. Thus assignments and arithmetic can be performed on it.

Syntax

The formal description of this primitive is the following: 
    ⟨tracingrestores⟩
       → \tracingrestores ...

Examples

    \tracingrestores=1
