This package contains primitives for ExTeX's eTeX mode.

The Count Parameter \eTeXversion

The count register \eTeXversion contains the version number of eTeX as an integer. The number is multiplied by 100 to show the fractional part. For instance the value 123 signals the version number 1.23 of eTeX.

The version number is a preassigned count register and can be used as such. Thus assignments and arithmetic can be performed on it.

Syntax

The formal description of this primitive is the following: 
    ⟨eTeXversion⟩
       → \eTeXversion {@linkplain
         de.dante.extex.interpreter.TokenSource#getOptionalEquals(Context)
         ⟨equals⟩} {@linkplain
         de.dante.extex.interpreter.TokenSource#scanInteger(Context,Typesetter)
         ⟨number⟩}

Examples

    \the\eTeXversion

The Tokens Parameter \eTeXrevision

The tokens register \eTeXrevision contains the revision of eTeX in use. Usually this is something like a letter.

The revision number is a preassigned tokens register and can be used as such. Thus assignments can be performed on it.

Syntax

The formal description of this primitive is the following: 
    ⟨eTeXrevision⟩
       → \eTeXrevision {@linkplain
         de.dante.extex.interpreter.TokenSource#getTokens(Context,TokenSource,Typesetter)
         ⟨tokens⟩}

Examples

    \the\eTeXrevision
