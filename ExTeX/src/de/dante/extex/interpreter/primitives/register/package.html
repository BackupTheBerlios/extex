Registers

This package contains primitives for the manipulation of registers. Registers are the primary means to store parameters which influence the behaviour or ExTeX as well as variables used in macro programming. The name of the registers can be a name or an number (between 0 and Integer.MAX_VALUE).

Several types of registers are provided:

\count

\count provides registers for integer values. The registers are numbered from \count0 up to \count0<Integer.Max_VALUE>. The count registers can be addressed by names, if a name has been specified by 

	\countdef\<name>=<register number>

\name may now be used as a synonym for \count<register number>.

The definitions are made in in the classes NumberedCount and NamedCount.

Example 

	\countdef\nose=40
	nose: \the\nose
	
	\count40=12
	count40: \the\count40

	nose: \the\nose

Output

nose: 0 The register \count40 ist not yet filled.
count40: 12 The register \count40 has been set to the value 12 in the line above.
nose: 12 \nose refers to \count40.

Version

Tested with NamedCount version 1.12 and NumberedCount version 1.8
